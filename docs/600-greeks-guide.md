# Греческие параметры опционов (Greeks)

> Что это, как считаются, когда увеличиваются/уменьшаются, как влияют на цену опциона.

---

## 📌 Общая концепция

**Greeks** — это производные функции цены опциона (по формуле Black-Scholes) по различным факторам:
- Цена базового актива
- Время до экспирации
- Волатильность
- Процентная ставка

Каждый Greek показывает **чувствительность** цены опциона к изменению одного фактора.

---

## 🔹 Delta (Δ)

**Что показывает:** Насколько изменится цена опциона при изменении цены базового актива на $1.

**Формула:**
```
Delta = ∂OptionPrice / ∂SpotPrice
```

**Для Put-опционов:** Delta отрицательная (обычно от -1.0 до 0).
**Для Call-опционов:** Delta положительная (обычно от 0 до 1.0).

### Когда растёт
- **Put:** при падении цены (ближе к ITM)
- **Call:** при росте цены

### Когда падает
- **Put:** при росте цены (удаляется от ITM)
- **Call:** при падении цены

### Влияние на цену опциона
```
ΔPrice ≈ Delta × ΔSpotPrice
```
*Пример:* Put Delta = -0.30, SOL вырос на $1 → Put подешевел на ~$0.30

### Влияние других факторов на Delta
| Фактор | Влияние на Delta Put | Влияние на Delta Call |
|--------|---------------------|---------------------|
| Рост цены | Растёт (→ 0) | Растёт (→ 1) |
| Увеличение IV | Растёт | Растёт |
| Уменьшение DTE (ближе к экспирации) | Уменьшается (→ 0 для OTM, → 1 для ITM) | Растёт (→ 1 для OTM, → 1 для ITM) |

### В контексте хеджа
- **Delta ≈ 0.05:** очень далёкий OTM (хвост)
- **Delta ≈ 0.10-0.20:** Anchor Layer (дальняя защита)
- **Delta ≈ 0.20-0.40:** Adaptation Layer (средняя защита)
- **Delta ≈ 0.40-0.60:** Active Layer (ATM/ближний)

## 🔹 Gamma (Γ)

**Что показывает:** Насколько изменится Delta при изменении цены на $1. Это **ускорение** Delta.

**Формула:**
```
Gamma = ∂Delta / ∂SpotPrice = ∂²OptionPrice / ∂SpotPrice²
```

### Когда растёт
- Максимум при **ATM** (Delta ≈ 0.5)
- При **коротком DTE** (ближе к экспирации)
- При **низкой IV** (цена более чувствительна к движению)

### Когда падает
- При движении вглубь **ITM** (Delta стабилизируется у 1.0 или 0)
- При **длинном DTE** (время смягчает чувствительность)
- При **высокой IV** (цена менее чувствительна)

### Влияние на цену опциона
```
ΔDelta ≈ Gamma × ΔSpotPrice
ΔPrice ≈ Delta × ΔSpotPrice + 0.5 × Gamma × (ΔSpotPrice)²
```

### В контексте хеджа
- **Gamma = "двигатель"** — когда Gamma растёт, Delta растёт ускоренно, позиция быстро становится прибыльной
- **Gamma = "тормоз"** — когда Gamma падает, Delta стабилизируется, движение замедляется

---

## 🔹 Theta (Θ)

**Что показывает:** Сколько стоит опцион каждый день (потеря стоимости из-за времени).

**Формула:**
```
Theta = ∂OptionPrice / ∂Days
```

**Для длинных позиций:** Theta всегда отрицательный (опцион теряет ценность с каждым днём).

### Когда растёт (усиливается)
- При **DTE < 30** (theta-распад ускоряется)
- При **ATM** (наибольшая временная стоимость)
- При **низкой IV** (меньше "воздуха" в опционе)

### Когда падает (слабеет)
- При **длинном DTE** (время "дёшево")
- При **высокой IV** (много временной стоимости)
- При **глубоком OTM/ITM** (меньше временной стоимости)

### Влияние на цену опциона
```
ΔPrice ≈ Theta × ΔDays
```
*Пример:* Theta = -$0.10/день → каждый день опцион теряет $0.10

### В контексте хеджа
- Theta — **стоимость содержания позиции**
- Anchor Layer: theta малый (длинный DTE)
- Active Layer: theta большой (короткий DTE, быстрая потеря)

---

## 🔹 Vega (ν)

**Что показывает:** Насколько изменится цена опциона при изменении IV на 1% (0.01).

**Формула:**
```
Vega = ∂OptionPrice / ∂IV
```

### Когда растёт
- При **длинном DTE** (больше времени для волатильности)
- При **ATM** (наибольшая чувствительность к IV)
- При **низкой базовой IV** (эффект сильнее)

### Когда падает
- При **кратком DTE** (мало времени)
- При **глубоком OTM/ITM** (меньше чувствительность)
- При **высокой IV** (эффект слабее)

### Влияние на цену опциона
```
ΔPrice ≈ Vega × ΔIV
```
*Пример:* Vega = 0.50, IV вырос на 5% (0.05) → цена опциона выросла на $0.25

### Влияние IV на Greeks
| Фактор | Delta | Gamma | Theta | Vega |
|--------|-------|-------|-------|------|
| Рост IV | ↑ (Put) | ↓ | ↑ (меньше убыток) | ↑↑ |
| Падение IV | ↓ (Put) | ↓ | ↓ (больше убыток) | ↓↓ |

---

## 🔹 Rho (ρ)

**Что показывает:** Чувствительность к процентным ставкам.

**В контексте проекта:** Пренебрежимо мало для крипто-опционов (ставки низкие).

| **Vega** | Чувств. к IV | ∂P/∂IV | Положит. | Длинный DTE, ATM | Короткий DTE, OTM |

## 🔄 Взаимосвязи Greeks

### Delta-Gamma
- **Gamma растёт → Delta растёт ускоренно** (идеально для прибыли)
- **Gamma падает → Delta стабилизируется** (прибыль замедляется)
- **Gamma = "двигатель", Delta = "скорость"**

### Theta-Gamma
- **Gamma растёт → Theta обычно растёт** (дороже содержать)
- **Gamma падает → Theta падает** (дешевле содержать, но и меньше прибыли)
- **Баланс:** пока Gamma × Delta > Theta → позиция выгодна

### Vega-Theta
- **Рост IV → Vega растёт, Theta уменьшается** (плюс к цене)
- **Падение IV → Vega падает, Theta растёт** (минус к цене)
- **IV и Theta враги:** высокая IV компенсирует theta-распад

---

## 📐 Влияние на цену опциона (полная формула)

```
ΔPrice ≈ Delta × ΔS + 0.5 × Gamma × (ΔS)² + Theta × Δt + Vega × ΔIV + Rho × Δr
```

Где:
- **ΔS** — изменение цены спота
- **Δt** — прошедшие дни
- **ΔIV** — изменение волатильности
- **Δr** — изменение ставок

**Для хеджа:** Delta — главный фактор, Theta — ежедневная стоимость, Gamma/Vega — ускорение/замедление.
