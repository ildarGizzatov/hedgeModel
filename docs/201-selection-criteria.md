# Критерии отбора опционов

> Правила фильтрации опционов для каждого слоя.

---

## Anchor Layer (Tail Risk)

> **Distance:** D = (S - K) / S, где S = текущая цена, K = страйк

| Параметр          | Диапазон           | Обоснование                              |
| ----------------- | ------------------ | ---------------------------------------- |
| Delta             | 0.05 – 0.20        | Глубоко OTM, дёшево, хеджирует хвост     |
| DTE               | >= 25              | Долгосрочная защита, меньше theta-расход |
| Distance от спота | ≥ 15%              | Хеджирование экстремальных сценариев     |
| Приоритет         | Efficiency (PnL/$) | Максимальный PnL на 1$ затрат            |

## Adaptation Layer (Stability)

| Параметр          | Диапазон           | Обоснование                        |
| ----------------- | ------------------ | ---------------------------------- |
| Delta             | 0.20 – 0.40        | Ближе к ATM, реагирует на движение |
| DTE               | 10 – 25 дней       | Среднесрочная стабилизация         |
| Distance от спота | D ≈ 5–10%          | Баланс между защитой и стоимостью  |
| Приоритет         | Drawdown Reduction | Максимальное снижение просадки     |

## Active Layer (Profit Engine)

| Параметр          | Диапазон    | Обоснование                              |
| ----------------- | ----------- | ---------------------------------------- |
| Delta             | 0.38 – 0.55 | ATM/близко к ATM, макс. чувствительность |
| DTE               | 5 – 10 дней | Быстрая реакция на движение              |
| Distance от спота | D ≈ 0–5%    | Ближайший слой, захват движения          |
| Приоритет         | Gamma (Γ)   | Максимальная Гамма                       |
