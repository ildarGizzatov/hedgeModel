## 🔹 Adaptation Layer (Средний)
> *Стабилизация PnL, плавный переход дельты, контроль просадки.*
**Статус:** ✅ Зафиксировано

**🟢 Держим, пока работают все 3 фактора:**
1. Цена движется (направленный импульс)
2. Gamma растёт (вход в зону максимальной чувствительности)
3. IV растёт (подкреплена ростом премии/Vega)

**🛑 Закрываем, когда импульс исчерпан:**
1. Цена замедлилась / боковик
2. Gamma реализована (delta ≈ 1.0 для ITM или падает для OTM)
3. IV стабилизировалась / упала
4. Theta-распад начал перевешивать

**⏱ Time Exit:**
- DTE < ? → проверяем экономику ролла ([[T003]])
- Ролл по горизонтали/вертикали ИЛИ полное закрытие

## 📊 Общие правила (Cross-Layer)
### Декомпозиция PnL
- **Воздушная прибыль (Vega/IV):** OTM, PnL > 0, intrinsic = 0 → не закрываем только на этом
- **Жёсткая прибыль (Delta):** ITM, PnL > 0, intrinsic > 0 → цель достигнута

### Триггеры выхода
- **PnL-based:** X% прибыли / Y% убытка
- **Time-based:** DTE < N
- **Delta/Gamma-based:** Ускорение закончилось / delta достигла потолка
- **IV-based:** IV упал ниже порога (прибыль уже зафиксирована)
