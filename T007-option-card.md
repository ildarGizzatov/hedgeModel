# T007 — Карточка опциона

> Создать стандартизированную карточку опциона — единый формат представления одного опциона для анализа LLM.

## 📌 Описание

Карточка опциона — структурированное представление всех ключевых данных по одному опциону, которое используется как входной контекст для принятия решений (выбор, мониторинг, закрытие).

## 📋 Объективные данные опциона

1. `symbol` — символ опциона
2. `DTE` — дни до экспирации
3. `strike` — страйк
4. `price` — цена
5. `delta` — дельта
6. `gamma` — гамма
7. `vega` — вега
8. `theta` — тета

## 📋 Рабочая область

Рабочая область опциона — диапазон цены базового актива, в котором опцион обеспечивает основную часть своей полезной защиты.

Определяется на основе кривой PnL через перегибы:

1. `zone_start` — начало рабочей зоны — когда прирост PnL начинает быстро увеличиваться (нижний перегиб кривой)
2. `zone_end` — конец рабочей зоны — когда прирост PnL начинает заметно замедляться (верхний перегиб кривой)
3. `zone_width` — ширина рабочей зоны (zone_end − zone_start)

- [ ] Определён формат вывода (текст / таблица / JSON)
