# 📋 T002 — Мониторинг тета-распада по каждому открытому опциону

> Отслеживать тета-распад для понимания величины временной стоимости.

## 📌 Описание

Создать систему, которая для каждого открытого опциона показывает:
- Сколько тета-стоимости «сгорает» каждый день
- Суммарный тета-распад по всему хедж-портфелю
- Накопленный тета-распад с момента покупки

## 🎯 Целевой слой

Мониторинг тета-распада полезен в первую очередь для **опционов среднего слоя (Adaptation Layer)**, так как его цель — стабилизация PNL.

## 📊 Вывод данных

- Ежедневный отчёт по каждому опциону в CSV/Excel
- Визуализация в Excel (график накопленного тета)
- Предупреждения при превышении порога тета-распада
